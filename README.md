## Tech Stack

- **Backend:** Django  
- **ML Tools:** sklearn, pandas, joblib, pickle  
- **Configuration Management:** python-decouple  
- **Messaging Service:** Twilio (for WhatsApp integration)  
- **Document Generation:** xhtml2pdf (for generating digital prescriptions)
